# user-auth-frontend
Frontend part of User Auth App, built in ReactJs with vite. It uses tailwindcss, shadcn-ui, lucid-react icons, etc.

## How to run this ReactJs frontend
Ensure you have .env file with ``` VITE_BASE_URL=http://localhost:8081/ ```

- Go to root of frontend project folder and run at command line =>
```
npm i
npm run build
npm run dev
```

- Now, find User-Auth-App project at http://localhost:5173
